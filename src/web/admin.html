종목 기준 대상 운영

검색 → 기존 대상 확인 → 인플레이스 저장/업데이트 → 결과 상태 확인 순서로 구성된 작업면입니다.

Step 1

종목 찾기

종목코드 또는 종목명으로 검색한 뒤 작업할 심볼을 선택합니다.

Step 2

현재 대상 확인

선택한 종목 + 시장 범위 조합에 이미 연결된 대상과 현재 런타임 상태를 먼저 확인합니다.

Step 3

대상 저장 / 업데이트

기존 대상이 있으면 새로 만들지 않고 같은 자리에서 업데이트합니다.

수집 이벤트
최소 1개 이상 선택
Step 4

결과 상태 확인

방금 저장한 대상이 전체 목록에서 어떤 상태인지 즉시 재확인합니다.

Secondary Job

이벤트 카탈로그

설정 가능한 이벤트 종류만 빠르게 확인하는 보조 뷰입니다.

Secondary Job

런타임 헬스

서비스 단위 상태와 오류를 메인 작업 흐름과 분리해서 확인합니다.

Secondary Job

스토리지 바인딩

연결된 저장소 바인딩 현황을 메인 대상 편집 작업과 분리해서 확인합니다.
